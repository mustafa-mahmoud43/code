# CodeOfLove

مشروع صفحة تفاعلية لطلب الخطبة — اسم المشروع: CodeOfLove

### تشغيل المشروع محليًا
1. افتح الطرفية في مجلد المشروع.
2. نفذ:
```bash
npm install
npm run dev
```
3. افتح المتصفح على العنوان الذي يظهر في الطرفية (عادة http://localhost:5173).

### ملاحظة عن الموسيقى
- ملف الموسيقى متوفر في `public/assets/bg-music.wav` (مضمَّن في الأرشيف).

### تنبيه
- المتصفحات تمنع التشغيل التلقائي للموسيقى؛ لذا يجب الضغط على زر "ابدأ الرحلة" لتشغيل الموسيقى على الموبايل.
